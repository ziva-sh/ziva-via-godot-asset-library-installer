# Ziva Installer

This folder is an auto-installer and updater for [Ziva](https://ziva.sh/), the only AI agent that lives inside the Godot editor.

## Need help or have ideas?

* Discord: https://discord.gg/hFAdBdmWxV (fastest)
* Docs: https://ziva.sh/docs
* Report a bug or request a feature: https://github.com/ziva-sh/ziva-agent-plugin-godot/issues

## Privacy

Code and prompts sent to Ziva is 100% private by default. When changing settings and models, refer to the 🛡️ (zero data retention), 🔒 (no training) and ❓ (other/unknown) icons to make an informed decision.


## What is Ziva?

Ziva is an AI copilot specialized for game development. Studios use it to ship all sorts of games, from 3D shooters to multiplayer games. Use it to generate assets, build scenes , or write code, from a selection of all the top LLMs.

## Uninstall

Close Godot, then Delete addons/ziva_agent and ziva_installer folders. Reopen Godot afterwards.
